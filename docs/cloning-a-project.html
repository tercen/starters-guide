Chapter 6 Cloning a project

In this section we clone (i.e. make a copy) the crabs project from tercen public projects and place it in your personal team.

  • Scroll down to tercen/crabs tutorial project
  • Click on tercen/crabs tutorial project

You can now see the chosen crab project and its files.

  • The icon for cloning is shown in the top right
  • Click on the cloning icon

A popup will show, giving you the option to clone to any team you’re in.

  • Click Ok

You now have cloned the project into your personal team.
